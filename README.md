# img2code

**img2code** allows you to convert images to codes :)

## About

What you need is just `Python` and `opencv-python` :)

To generate an image, use this command:

```sh
python img2code.py <Image Path> <Name of output file> <Language>
```

- Notice `<Name of output file>` and `<Language>` are **optional**. Default output file is: `output.txt`
- The language actually only affects the *comments* type like `/* */` in *C*++ and `{- -}` in *Haskell*.

Here is an example:

```sh
python3 img2code.py "ruler.jpg" "ruler" "hs"
```
